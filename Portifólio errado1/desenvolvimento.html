Desenvolvimento WEB

O que é?

O desenvolvimento web é o processo de criação, construção e manutenção de websites e aplicações web. Ele envolve diversas disciplinas e habilidades, abrangendo desde o design de interfaces visuais até a programação de funcionalidades complexas.


Principais Componentes do Desenvolvimento Web


1. Frontend Development (Desenvolvimento do Lado do Cliente):

° HTML (HyperText Markup Language): Linguagem de marcação usada para estruturar o conteúdo da web.

° CSS (Cascading Style Sheets): Linguagem de estilos usada para controlar a aparência visual do conteúdo HTML.

°JavaScript: Linguagem de programação usada para adicionar interatividade e funcionalidades dinâmicas aos websites.


2. Backend Development (Desenvolvimento do Lado do Servidor):

° Linguagens de Programação: Como Python, Java, PHP, Ruby, Node.js, entre outras, usadas para construir a lógica do servidor e manipular dados.

° Bancos de Dados: Sistemas de gerenciamento de banco de dados, como MySQL, PostgreSQL, MongoDB, usados para armazenar e recuperar dados.

° Servidores: Software e hardware que hospedam o site e servem seu conteúdo aos usuários (por exemplo, Apache, Nginx).


3. Full Stack Development:

° Envolve habilidades tanto de frontend quanto de backend, permitindo que o desenvolvedor crie e gerencie todas as partes de um aplicativo web.

Linguagens e Conteúdos



HTML

HTML, ou HyperText Markup Language (Linguagem de Marcação de Hipertexto), é a linguagem fundamental usada para criar e estruturar conteúdo na web. HTML utiliza uma série de elementos, conhecidos como tags, que indicam ao navegador como exibir o conteúdo da página.


Estrutura Básica de um Documento HTML





Principais Elementos HTML

DOCTYPE html:
Declara o tipo de documento e a versão do HTML. Em HTML5, é simplesmente !DOCTYPE html>.


html:
Elemento raiz que engloba todo o conteúdo da página.


head:
Contém metadados sobre o documento, como seu título, codificação de caracteres, e links para estilos e scripts.


title:
Define o título da página que aparece na aba do navegador.


meta:
Define metadados, como charset para especificar a codificação de caracteres e viewport para controle da responsividade em dispositivos móveis.


body:
Contém todo o conteúdo visível da página web.


CSS

CSS (Cascading Style Sheets), ou Folhas de Estilo em Cascata, é uma linguagem utilizada para descrever a apresentação de documentos escritos em HTML ou XML. CSS controla o layout e a aparência visual das páginas web, permitindo que desenvolvedores separem o conteúdo (HTML) da apresentação (CSS).

JavaScript

JavaScript é uma linguagem de programação de alto nível, interpretada e orientada a objetos, amplamente utilizada para criar interatividade e dinamismo em páginas web. Ela é uma das tecnologias fundamentais da web moderna e é suportada por todos os navegadores web populares. JavaScript desempenha um papel fundamental no desenvolvimento web moderno, permitindo a criação de páginas e aplicativos interativos e dinâmicos. Seu uso é essencial para criar experiências ricas para o usuário na web e além dela.


Principais Características do JavaScript

Linguagem de Script:
JavaScript é uma linguagem de script, o que significa que o código é executado linha por linha, conforme é encontrado, sem a necessidade de compilação prévia.

Interpretada:
O código JavaScript é interpretado pelo navegador web em tempo de execução.

Orientada a Objetos:
JavaScript suporta programação orientada a objetos, permitindo a criação de objetos, métodos e propriedades para organizar e manipular dados.

Multiplataforma:
JavaScript é suportado por todos os principais navegadores web e pode ser executado em diferentes sistemas operacionais.

Principais Conceitos do JavaScript

Variáveis e Tipos de Dados: Declaração de variáveis e manipulação de diferentes tipos de dados, como strings, números e arrays.

Funções: Declaração e chamada de funções para executar tarefas específicas.

Estruturas de Controle: Uso de estruturas como if, else, for e while para controlar o fluxo do programa.

Manipulação do DOM: JavaScript é usado para interagir com o Document Object Model (DOM) de uma página web, permitindo alterações dinâmicas na estrutura e no conteúdo da página.

Eventos: Manipulação de eventos, como cliques do mouse e pressionamentos de tecla, para criar interatividade.
